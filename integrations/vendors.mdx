title: "Supported Vendors"
description: "Langtrace supports a variety of vendors and integrations."
---

Langtrace automatically captures traces from the following vendors:

| Vendor       | Type            | Typescript SDK | Python SDK |
| ------------ | --------------- | -------------- | ---------- |
| OpenAI       | LLM             | &#x2714;       | &#x2714;   |
| Anthropic    | LLM             | &#x2714;       | &#x2714;   |
| Azure OpenAI | LLM             | &#x2714;       | &#x2714;   |
| Langchain    | Framework       | &#x2718;       | &#x2714;   |
| LlamaIndex   | Framework       | &#x2714;       | &#x2714;   |
| Pinecone     | Vector Database | &#x2714;       | &#x2714;   |
| ChromaDB     | Vector Database | &#x2714;       | &#x2714;   |

We also support any data visualization platform that supports OpenTelemetry, these include: Grafana, Dynatrace, DataDog, and more.

If you would like to see support for a specific vendor, please [contact us](https://calendar.app.google/Go5gXNPcqZjAY4i47) and let us know!
