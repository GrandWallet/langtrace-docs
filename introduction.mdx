## Setting up

The first step to using our SDK is to import into your Typescript or Python project.

Step 1. Generate an API key from the Langtrace AI dashboard.

- Go to the [Langtrace AI dashboard](https://langtrace.ai/login)
- Click on the `API Keys` tab
- Click on the `Generate API Key` button
- Copy the generated API key

Step 2. Install the SDK on your project:

<CodeGroup>

```typescript Typescript
import { init } from "@langtrace-init/init";

init({ api_key: LANGTRACE_API_KEY });
```

```python Python
from langtrace_python_sdk import langtrace

langtrace.init(api_key=LANGTRACE_API_KEY)
```

</CodeGroup>

Step 3. Review your traces on the Langtrace AI dashboard. You can now view your traces on the [Langtrace AI dashboard](https://langtrace.ai/dashboard).
